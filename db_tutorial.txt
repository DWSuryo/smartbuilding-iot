how to modify db in SQLAlchemy:
in terminal:

>>> python
>>> from hello import db
>>> db.create_all               #creates site.db
>>> from hello import User, Post        #imports module     
>>> user_1 = User(username='Admin', email='admin@admin.com', password='admin')  #asssign user_1
>>> db.session.add(user_1)      #adds user (not added to db yet)

can add more user (step 7 and 8)
commit:

>>> db.session.commit()

find user:
>>> User.query.all()
>>> User.query.filter_by(username='Admin').first()
>>> user=User.query.filter_by(username='Admin').first()
user.username
user.id
user = User.query.get(1)
user.posts
...

delete all SQL data:
>>> db.drop_all()           deletes (drops) the tables
>>> db.create_all()         create new table
>>> User.query.all()        check users
>>> Post.query.all()        check posts
